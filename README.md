# CGPA_Calculator
A simple Java based GUI app to calculate the CGPA of students, using NetBeans IDE. It is connected with the MySQL Database to fetch the subjects of the branch as per applicable.

* Language: Java
* IDE: NetBeans
* Database System: MySQL
